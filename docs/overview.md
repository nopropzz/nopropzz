# NOPROPZZ / SYSTEM_OVERVIEW
VERSION: 1.0.0
STATUS: ACTIVE

## 01. VISION
NoPropzz is a creative collective redefining Nordic design through raw brutalism. The web platform serves as both a public archive (Portfolio, Talent, Radio) and an operational Command Center (Admin Dashboard).

## 02. DESIGN PHILOSOPHY: NORDIC BRUTALISM
- **Honesty in Materials**: Pixels and code are treated as raw materials. No unnecessary shadows or gradients.
- **High Contrast**: Pure White (#FFFFFF) and Deep Onyx (#222222).
- **Typography**: Archivo (Heavy/Black) for impact; Courier Prime (Mono) for data and technical details.
- **Asymmetry**: Purposeful layout tension to drive user engagement.

## 03. TECHNICAL STACK
- **Frontend**: React 19 (ESM)
- **Styling**: Tailwind CSS (Utility-first)
- **Animation**: Framer Motion (Orchestration)
- **Icons**: Lucide React
- **Analytics**: Recharts
- **Routing**: React Router 7 (Hash-based)

## 04. CORE MODULES
- **Portfolio**: Categorized archival display of visual work.
- **Radio**: Minimalist audio interface for studio mixes.
- **Talent Division**: Representation roster for Models and Artists.
- **Admin Gateway**: Secure dashboard for content management and analytics.
